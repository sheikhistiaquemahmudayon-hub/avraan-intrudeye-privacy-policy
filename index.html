About IntrudEye

IntrudEye is a security and anti-theft application developed by AVRAAN. The application helps users protect their devices from unauthorized access, monitor security events, and improve device safety through intelligent protection features.

Permissions We Use

📷 Camera Permission

Used to capture a photo when an unauthorized unlock attempt is detected for security purposes.

📍 Location Permission

Used to help locate a lost or stolen device and support anti-theft functionality.

🛡 Device Administrator

Required to provide advanced security monitoring and device protection features.

♿ Accessibility Service

Used exclusively for security-related functionality such as unauthorized access detection and lock-screen monitoring. No personal content is collected through this service.
